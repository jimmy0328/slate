    這是 ImVIP API 使用文件手冊，開放給 POS 整合專用

    認證

    認證的使用方式

    HTTP Request

    POST http://api.imvip.io:9002/login

    data

    變數 TYPE 範例
    username 字串 Nick
    password 字串 coffeeshop12345677

    response

    變數 值 範例
    access_token basic ${token} basic 3979723hjjhdshfdskjhfhkjsdfjkdkjf8y423
    請注意，access_token 在每一個 request 都必須放置在 header 一起送出，否則認證則會失敗

    取得活動設定

    $.post('http://api.imvip.io:9002/login', {
    username: '${username}',
    password: '${password}'
  }).done(function (res) {
   var response = JSON.parse(res);
   console.warn(response.access_token)
})
